# Setup

1. Place the ML_Echoes.jar file into your Hytale mods folder.
2. Create a world with the ML_Echoes mod enabled, or right-click an existing world to edit its settings and enable the mod.

----

# Settings

* To open the plugin menu, run this command in chat:<br>
